# Grotrack

A Progressive Web App for managing your home grocery inventory.

## Features

- Track items across compartments (Fridge, Freezer, Pantry, etc.)
- Automatic expiration date tracking
- Search your inventory
- Mobile-friendly design

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Zustand (state management)
- Vite (build tool)
